# Demo images

Drop `image1.jpg` … `image5.jpg` here for the photo slideshow in
`page_demo.py` (Home → Photos). Any Pillow-readable format works; rename to
`imageN.jpg` or adjust the paths in `page_demo.py`.

Until they exist, the slideshow shows a "broken image" placeholder per slide —
the page still renders (drm_composer treats a missing `<img src>` leniently).
